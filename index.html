Visualizing World Weather

Weather, It's all Around Us!

"Whether" it is the focal point of polite conversation, part of planning a trip or figuring out what clothes to wear, atmospheric conditions are a ubiquitous part of everyday life. Seems like this makes it a great topic for some data analysis, so this site has a number of graphs analyizing temperature, humidity, wind and cloudiness in relation to latitude.

Global Meteorology

This site contains graphical renderings based on information extracted using the Open Weather Map API which catalogs meteorological conditions world wide. The Google map above shows an analysis of humidity levels from the data, which can be examined by accessing the link above.

The scripts for pulling information from the Open Weather Map API and evaluating the data using the Python are on Github. Repository here: Weather Analysis. Also on GitHub are the image files and code for this website: World Weather Visualization.
