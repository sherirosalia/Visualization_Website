Illustrating the Weather of the World

Data analysis and visualization using programming.

Most of life's adventures are more fun when shared. Data is no different and this website is the end result of multiple programming skills utilized to convert raw data into an easy to understand format.

In order to illustrate the above this project uses data science in an attempt to prove and possibly challenge the widely held assumption that global temperatures rise in proximity to the equator. In addition, humidity, cloud cover percentages and windspeed readings were also included to counter check correlations.

In order to accomplish the above, python programming choreographed with the Open Weather API and the CityPy library was utilized to compile the dataset. Visualizations were created using Pandas and Matplotlib from the Python library. The end results displayed here were handcoded in HTML, CSS and Bootstrap was incorporated to ensure multiple device responsiveness.
