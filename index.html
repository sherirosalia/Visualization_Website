Illustrating the Weather of the World

Data analysis and visualization using programming.

Data science applied to the assumption that global temperatures rise in proximity to the equator. Analysis of humidity, cloud cover percentages and windspeed readings were also included to counter check correlations.

The data was pulled on Open Weather Map using the Python API and the companion repository is here: Weather Analysis and the repository for this site is here: World Weather Visualization.

The visualizaton at the left reflects the most humid and dry regions of the world, green having the highest humidity and red the lowest. Since the data points are coordinated with international city geolocations, the general shape of the plot matches a world map.
